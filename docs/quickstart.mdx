# Quickstart

Get Sukko running locally and receive your first real-time message in under 5 minutes.

:::info Time to complete
~5 minutes
:::

## Prerequisites

- **Docker** installed and running ([Install Docker](https://docs.docker.com/get-docker/))
- **A terminal** (macOS, Linux, or Windows WSL2)

## Step 1: Install the Sukko CLI

Verify the installation:

```bash
sukko version
```

Expected output:

```
sukko version v0.x.x (commit: abc1234)
```

## Step 2: Start Sukko

```bash
sukko up
```

This pulls Docker images and starts the full Sukko stack:
- **ws-gateway** — WebSocket reverse proxy (port 3000)
- **ws-server** — Core message server
- **provisioning** — Multi-tenant management API (port 8080)
- **Redpanda** — Kafka-compatible message broker
- **NATS** — Inter-pod broadcast bus

Wait for all services to be healthy (~30 seconds):

```
✓ All services healthy
  Gateway:      ws://localhost:3000
  Provisioning: http://localhost:8080
```

## Step 3: Create a Tenant

Every connection in Sukko belongs to a tenant. Create one:

```bash
sukko tenant create --id demo --name "Demo Tenant"
```

Expected output:

```json
{
  "id": "demo",
  "name": "Demo Tenant",
  "status": "active"
}
```

## Step 4: Generate a Token

Clients authenticate with JWT tokens. Generate one for testing:

```bash
sukko token generate --tenant demo --subject user-1
```

Copy the token — you'll use it in the next step.

## Step 5: Subscribe to a Channel

Open a second terminal and subscribe to a channel:

```bash
sukko subscribe general.chat --tenant demo --subject user-1
```

You'll see:

```
Connected to ws://localhost:3000
Subscribed to: general.chat
Waiting for messages... (Ctrl+C to exit)
```

## Step 6: Publish a Message

In your first terminal, publish a message:

```bash
sukko publish general.chat '{"text": "Hello, Sukko!"}' --tenant demo
```

In the subscriber terminal, you'll see:

```json
{
  "channel": "general.chat",
  "data": {"text": "Hello, Sukko!"}
}
```

You just sent your first real-time message through Sukko.

## Step 7: Clean Up

When you're done:

```bash
sukko down
```
