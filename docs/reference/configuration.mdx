# Configuration Reference

All configurable environment variables across Sukko services. Values are set via `env:` struct tags in Go config files â€” the `envDefault` is the source of truth for defaults.

## Shared (Base)

| Variable | Type | Default | Description |
|----------|------|---------|-------------|
| `LOG_LEVEL` | string | `info` | Logging level (debug, info, warn, error). info is recommended for production. |
| `LOG_FORMAT` | string | `json` | Log output format: json for machine parsing (production), text for human-readable (development). |
| `ENVIRONMENT` | string | `local` | Deployment environment name (e.g. local, dev, stag, prod). Used for Kafka topic namespace resolution. |
| `SUKKO_LICENSE_KEY` | string | â€” | Sukko license token (JWT). Community edition has no key; Pro/Enterprise obtain via Sukko account. Leave empty for Community. |
| `OTEL_TRACING_ENABLED` | bool | `false` | Enable distributed tracing via OpenTelemetry. Disabled by default; configure OTEL_EXPORTER_TYPE and OTEL_EXPORTER_ENDPOINT when enabled. |
| `OTEL_EXPORTER_TYPE` | string | `otlp-grpc` | OpenTelemetry trace exporter type: otlp-grpc, otlp-http, or stdout. Required when OTEL_TRACING_ENABLED=true. |
| `OTEL_EXPORTER_ENDPOINT` | string | `localhost:4317` | OpenTelemetry collector endpoint (e.g. otelcol:4317). Required when OTEL_EXPORTER_TYPE is otlp-grpc or otlp-http. |
| `PPROF_ENABLED` | bool | `false` | Enable Go pprof profiling endpoints (/debug/pprof/). Disabled by default; enable only in controlled environments. |
| `PYROSCOPE_ENABLED` | bool | `false` | Enable continuous profiling via Pyroscope. Disabled by default. |
| `PYROSCOPE_ADDR` | string | `http://localhost:4040` | Pyroscope server address (e.g. http://pyroscope:4040). Required when PYROSCOPE_ENABLED=true. |
| `AUTH_MODE` | string | `required` | Authentication enforcement mode. Only "required" is supported; tokens without valid signatures are rejected. |
| `PROVISIONING_GRPC_ADDR` | string | `localhost:9090` | gRPC address of the provisioning service for internal communication (e.g. channel rule streaming). |
| `HTTP_READ_TIMEOUT` | time.Duration | `15s` | Maximum duration for reading a complete HTTP request. Protects against slow-loris attacks. |
| `HTTP_WRITE_TIMEOUT` | time.Duration | `15s` | Maximum duration for writing the HTTP response. |
| `HTTP_IDLE_TIMEOUT` | time.Duration | `60s` | Maximum idle time on a keep-alive connection before it is closed. |
| `KAFKA_TOPIC_NAMESPACE_OVERRIDE` | string | â€” | Override for the Kafka topic namespace (normally derived from ENVIRONMENT). Use with care in production deployments. |
| `VALID_NAMESPACES` | string | `local,dev,stag,prod` | Comma-separated list of allowed Kafka topic namespaces. Used to validate KAFKA_TOPIC_NAMESPACE_OVERRIDE. |
| `MESSAGE_BACKEND` | string | `direct` | Message ingestion backend: direct (no persistence, no Kafka dependency) or kafka (full Kafka/Redpanda integration with replay). |
| `KAFKA_BROKERS` | string | `localhost:19092` | Comma-separated Kafka/Redpanda broker addresses for ws-server and tester only (provisioning uses PROVISIONING_KAFKA_BROKERS instead); required when MESSAGE_BACKEND=kafka. |
| `KAFKA_SASL_ENABLED` | bool | `false` | Enable SASL authentication for Kafka connections. Required for most managed Kafka/Redpanda services. |
| `KAFKA_SASL_MECHANISM` | string | â€” | SASL mechanism: scram-sha-256 or scram-sha-512. Required when KAFKA_SASL_ENABLED=true. |
| `KAFKA_SASL_USERNAME` | string | â€” | SASL username for Kafka authentication. Required when KAFKA_SASL_ENABLED=true. |
| `KAFKA_SASL_PASSWORD` | string | â€” | SASL password for Kafka authentication. Required when KAFKA_SASL_ENABLED=true. |
| `KAFKA_TLS_ENABLED` | bool | `false` | Enable TLS encryption for Kafka connections. Required for most managed Kafka/Redpanda services. |
| `KAFKA_TLS_INSECURE` | bool | `false` | Skip TLS certificate verification. For development only â€” never use in production. |
| `KAFKA_TLS_CA_PATH` | string | â€” | Path to CA certificate file for verifying the Kafka broker's TLS certificate. |
| `DATABASE_URL` | string | â€” | PostgreSQL connection URL (postgres://user:pass@host:5432/db). Required for services that use the database (provisioning, push). |

| `PUSH_GRPC_ADDR` | string | `localhost:3008` | Push service gRPC address for device registration and VAPID key endpoints. |
| `SSE_KEEPALIVE_INTERVAL` | time.Duration | `45s` | Interval for SSE keepalive messages sent to prevent proxy timeouts on long-lived connections. |
